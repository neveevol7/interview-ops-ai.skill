### 🎯 核心价值 / Core Value
这是一个专为 **Product Manager / Project Manager** 设计的面试辅助 Skill。通过将资深 PM 的方法论工程化，它能帮你快速生成一份具备“确定性交付价值”的面试攻略。

---

### 🚀 快速上手 (Quick Start)

1. **获取指令**：复制本仓库中 [`core-engine.md`](./core-engine.md) 的全部内容。
2. **运行 Skill**：将内容发送给 **Gemini, ChatGPT 或 Claude**。
3. **输入数据**：按照 AI 引导提供简历、JD、面试轮次及薪资。
