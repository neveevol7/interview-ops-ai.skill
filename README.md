# Interview-Ops-AI 🚀

> **面试不是一场考试，而是一次基于价值对等的产品交付。**

`Interview-Ops-AI` 是一个基于 AI 驱动的面试深度适配引擎。它将资深产品经理（PM）的思维逻辑与项目管理（NPDP）体系相结合，通过结构化的输入，自动生成一份具备“确定性交付价值”的面试通关攻略。

### 🌟 核心特性
- **ROI 驱动逻辑**：不谈感情，只谈你能为公司节省多少“试错成本”与“磨合时间”。
- **项目对标矩阵**：自动将你的简历项目与 JD 关键词进行 STAR 匹配。
- **动态薪资博弈**：根据当前薪资与 JD 范围，自动计算谈判筹码与话术。
- **全轮次适配**：针对 HR 面、业务面、终面，自动调整输出侧重点。

### 🛠️ 如何使用
1. 复制本仓库中的 `core-engine.md` 内容。
2. 发送给任何支持长文本的 AI（如 Gemini, Claude, GPT-4）。
3. 按照 AI 的引导，依次输入你的简历、JD 及薪资等信息。
4. 获取一份定制化的 `.md` 面试攻略。
